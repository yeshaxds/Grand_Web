# Vue 项目模板

## 项目设置
```
npm install
```

### 开发环境编译和热重载
```
npm run serve
```

### 生产环境编译和压缩
```
npm run build
```

### 代码检查和修复
```
npm run lint
```

## 项目结构
```
src/              # 源代码
  ├── assets/     # 静态资源
  ├── components/ # 公共组件
  ├── router/     # 路由配置
  ├── store/      # Vuex状态管理
  ├── views/      # 页面视图
  ├── App.vue     # 根组件
  └── main.js     # 入口文件
public/           # 公共资源目录
  └── index.html  # HTML模板
```
